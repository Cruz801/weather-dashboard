# weather-dashboard

This applicaiton is used to find current weather conditions and the future weather outlook for multiple cities. It also Includes a search history so that users can access their past searched cities.

# Technologies Used
* JQuery
* MomentJs
* BootStrap
* JavaScript
* CSS
